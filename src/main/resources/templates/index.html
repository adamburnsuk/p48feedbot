Device Management Tool

The QE Enablement Device Management tool is used to remotely control and configure the devices in the QLab used for test automation. You will find a list of devices, along with various ways to view device information and interact with the devices themselves.

Rest API

Anything that you can do through the UI can also be done through REST service calls as well. Click on the button below to access a list of devices that are currently available.
